Boardmaker

Python 3 Module

Chart with Ease

Create 2D lists that can be manipulated and displayed in a nice, tidy format.

Useful for spreadsheets, charts, games, user-interactions, and an overall aesthetically pleasing layout.

makeBoard class

  • Create an empty 2D list with custom dimensions >>>my_board = makeBoard(3,3)

What does it do?

-Automatic formatting for printing the data in an even layout.
